# Immersive 3D Learning

Immersive 3D Learning is an open-source platform that delivers immersive, curriculum-driven learning in a multiplayer 3D world. Learners join as avatars, complete interactive nodes, and track progress while developers extend the system via a modular TypeScript frontend and Spring Boot backend.

Features
- Avatar-based learning in a browser 3D world
- Node/curriculum-driven lessons
- Real-time presence and collaboration
- TypeScript + Vite frontend and Spring Boot backend

Getting started
1. Install Node.js and Java 17
2. From the workspace root, install frontend dependencies and run the dev server:

```bash
npm install
npm run dev
```

3. To run the backend, open the `api` folder and use Gradle or Maven as configured.
